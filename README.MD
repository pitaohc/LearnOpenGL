# OpenGL项目模板

## 方法1
把Release中的OpenGL Template.zip放到以下两个目录

1. C:\Users\xxx\Documents\Visual Studio 2022\My Exported Templates
2. C:\Users\xxx\Documents\Visual Studio 2022\Templates\ProjectTemplates

其中C:\Users\${Username}\Documents\${Visual Studio}可能需要修改。
${Username}为用户名称，${Visual Studio}为VS文档目录，如果是2022版本则为Visual Studio 2022

## 方法2
启动工程，生成模板
